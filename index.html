An open source reporting tools for Ruby

Thinreports は PDF 形式の帳票を生成する Ruby 向けのオープンソースソフトウェアです。 専用のレポートデザイナ Thinreports Editor と PDF ジェネレータ Thinreports Generator によって Ruby や Ruby on Rails を使った帳票開発の負担を軽減 - thin - することを目指します。
