公式サイトを GitHub Pages へ移行しました

2014.07.23 / written by @hidakatsuya

公式サイト www.thinreports.org を GitHub Pages へ完全に移行しました。

移行した理由は以下の通りです。

  1. デプロイを自動化し @hidakatsuya のような特定のユーザだけでなく、コミュニティによる更新が可能な環境へ
  2. 旧ホスティングサーバのメンテナンス業からの解放

www.thinreports.org は、 Ruby 製静的サイトジェネレーター nanoc によって管理されており、 そのソースコードは GitHub にて公開しています。サイトのリンク切れや間違いを見つけたら こちら よりご報告いただけると嬉しいです。もちろん Pull Request もお待ちしております（近く、ローカル環境でのサイト開発、確認方法を含めた Pull Request 手順をまとめます）

また、サイトのソースコードや更新履歴は thinreports.github.io にて確認できます。
