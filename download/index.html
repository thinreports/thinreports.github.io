Thinreports Editor 0.8.0 2015.04.30

  • Release Announcement
  • Changelog
  • Chrome Web Store
  • Installation Guide

Thinreports Editor は Chrome アプリケーションです。Chrome Web Store よりインストールできます。 利用するためには Chrome ブラウザが必要です。利用可能な OS や必要なソフトウェア、 インストール手順については Installation Guide を確認してください。 従来のデスクトップバージョンは、Windows 版のみリリースしています。こちら からダウンロードできます。

Thinreports Generator for Ruby 0.8.0 2015.04.30

  • Release Announcement
  • Changelog
  • RubyGems.org
  • Installation Guide

Generator for Ruby は RubyGems です。インストールは基本的に gem コマンドで行いますが、 rubygems.org/thinreports より パッケージファイルをダウンロードすることも可能です。詳細については Installation Guide をご覧ください。
